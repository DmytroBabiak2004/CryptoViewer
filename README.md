CryptoViewer — це невеликий WPF-застосунок на .NET 9 для перегляду інформації про криптовалюти. Дані отримуються через публічне API CoinGecko.

Що вміє програма

* Показує список популярних криптовалют.
* Дозволяє шукати монети по всій базі CoinGecko.
* Відкриває детальну інформацію про вибрану монету:

  * поточна ціна;
  * зміна ціни;
  * ринкова капіталізація;
  * обсяг торгів;
  * графік ціни за останні 7 днів;
  * список бірж, де торгується монета.
* Є конвертер між криптовалютами.
* Можна переключати світлу та темну тему.
* Є українська та англійська мови.

## Що використовував

* .NET 9
* WPF
* OxyPlot (для побудови графіків)
* HttpClient
* System.Text.Json
* MVVM

## Структура проєкту


Models      - моделі даних
Services    - робота з API, темами та мовою
Interfaces  - інтерфейси сервісів
ViewModels  - логіка сторінок
Views       - вікна та сторінки програми


Як працює

Програма отримує дані з CoinGecko через HTTP-запити.

Для того щоб не робити занадто багато однакових запитів, результати тимчасово зберігаються в кеші. Це допомагає швидше завантажувати дані та не перевищувати обмеження API.

Як запустити

1. Відкрити `CryptoViewer.sln` у Visual Studio 2022 або новішій.
2. Запустити проєкт.
3. Для роботи потрібне підключення до Інтернету.

Обмеження

* Використовується безкоштовне API CoinGecko, тому при дуже великій кількості запитів можуть з'являтися обмеження.
* Конвертер працює тільки з топ-50 криптовалют.
